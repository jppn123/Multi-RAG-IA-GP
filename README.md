# Multi-RAG Generative AI

Projeto acadêmico desenvolvido para a disciplina de Gestão de Projetos, com o objetivo de planejar, organizar e acompanhar o desenvolvimento de uma solução baseada em Inteligência Artificial Generativa utilizando a arquitetura Multi-RAG (Multiple Retrieval-Augmented Generation).

A solução tem como finalidade oferecer suporte a diferentes áreas organizacionais, incluindo:

- Atendimento ao cliente;
- Suporte técnico;
- Operações internas;
- Apoio à tomada de decisão.

O projeto contempla atividades de levantamento de requisitos, definição da arquitetura, planejamento do cronograma, gerenciamento de tarefas, acompanhamento de entregas e simulação da execução do projeto utilizando GitHub Projects como ferramenta de gestão.

## Objetivos

- Aplicar práticas de gerenciamento de projetos em um ambiente real.
- Organizar backlog, cronograma e entregas utilizando GitHub Projects.
- Simular o ciclo de vida de um projeto de tecnologia.
- Utilizar metodologias ágeis para acompanhamento e controle das atividades.
- Desenvolver competências relacionadas à colaboração, planejamento e gestão visual de projetos.

## Equipe

Este projeto é desenvolvido pela equipe da disciplina, sendo cada integrante responsável por atividades específicas definidas no planejamento do projeto.

## Ferramentas Utilizadas

- GitHub
- GitHub Projects
- GitHub Issues
- Git
- Inteligência Artificial Generativa
- Arquitetura Multi-RAG

## Estrutura do Projeto

O gerenciamento do projeto está organizado através de:

- Backlog de atividades;
- Quadro Kanban;
- Milestones (marcos do projeto);
- Cronograma de execução;
- Controle de responsáveis;
- Acompanhamento do progresso das entregas.
